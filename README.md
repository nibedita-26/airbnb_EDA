# airbnb_EDA
## Project Type
Exploratory Data Analysis (EDA)

## Objective
Analyze pricing, availability, and demand patterns across NYC Airbnb listings.

## Dataset
Airbnb NYC 2019 dataset containing listing-level information.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Analysis
- Univariate, Bivariate, and Multivariate Analysis
- Feature Engineering (Occupancy & Revenue Proxy)
- Data Cleaning and Outlier Handling
